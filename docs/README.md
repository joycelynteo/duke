# User Guide

## Features 

### Feature 1: list
List all the tasks.
### Feature 2: add
Add tasks to the list. There are 3 kinds of tasks:
1. Todo
2. Deadline
3. Event
### Feature 3: done
Mark task as completed.
### Feature 4: delete
Remove a task from the list.
### Feature 5: undo
Undo previous command.
### Feature 6: bye
Close the application.
